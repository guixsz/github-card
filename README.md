# GitHub Card

Sistema desenvolvido para consultar dados públicos de usuários do GitHub em tempo real, utilizando integração direta com a API oficial.

##  Funcionalidades

- Conexão com Api do [GitHub]("https://github.com/")
- Busca de dados por JSON
- Tratamento de erros
- Validação de dados


## 🛠️ Tecnologias e Ferramentas

* JavaScript
* HTML5 & CSS3
* GitHub REST API

## ⚠️ Restrições da API

- O GitHub impõe um limite de requisições para IPs não autenticados. Caso você receba um erro **403 Forbidden**, a cota de buscas temporária foi excedida. 
> **Nota:** Aguarde alguns minutos ou utilize uma conexão autenticada para normalizar o serviço.
